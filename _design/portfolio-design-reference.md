# デザインリファレンス分析 — ポートフォリオサイト

> Notionから取得: 2026-05-30  
> 参考サイト3つの特徴を詳細に分析し、ポートフォリオへの応用方針をまとめたドキュメント。

## 1. M-TRUST CO.,LTD. — https://m-trust.co.jp/

**受賞歴:** CSS Design Awards / Special Kudos 2024（UI 7.97 / UX 8.06 / Innovation 7.99）  
**制作:** Yuto Takahashi / LIG agency  
**タグ:** animated, liquid, WebGL

### ビジュアル・カラー
- ダーク系背景（黒〜深いネイビー）をベースに、明暗コントラストを極限まで高めた配色
- テキストはほぼ白のみ。アクセントカラーは最小限
- フルスクリーンを占有するビジュアルが基本単位

### タイポグラフィ
- 日本語・英語の大きなディスプレイフォントを大胆に配置
- テキスト自体がビジュアルエレメントとして機能
- 行間・字間を広く取り、余白で「重み」を演出

### アニメーション・インタラクション
- **最大の特徴: WebGL + DOM Video を組み合わせたパーティクルエフェクト**（動画がパーティクル爆発するエントランス演出）
- スクロールに連動したパーティクルの変形・分散
- リキッドモーフィング（液体が変形するような曲線アニメーション）
- 要素のフェードイン・スライドインはスクロールトリガー

### レイアウト
- 縦長・フルスクリーンセクションの積み重ね
- グリッドより「余白の設計」を優先したフリーレイアウト
- セクション間の境界を曖昧にするオーバーラップ手法

### ローディング
- 独自のローディングアニメーションあり（即座に世界観に引き込む）

## 2. Laugh Mind Co.,Ltd. — https://laugh-mind.co.jp/

**受賞歴:** Awwwards Honorable Mention 2025年5月 / CSS Design Awards  
**制作:** Laugh Mind Co.,Ltd. + Yuto Takahashi  
**タグ:** Scrolling, Transitions, Interaction Design

### ビジュアル・カラー
- **カラーパレット: `#c9e936`（蛍光イエローグリーン）× `#111111`（ほぼ黒）**
- ダーク背景に蛍光アクセントという強烈なコントラスト
- 写真・動画は最小限、タイポグラフィとグラフィックが主役

### タイポグラフィ
- 大胆な欧文サンセリフ（極太〜極細の混在）
- 日英混在テキストを縦横に大きく配置
- テキストがアニメーションの主体（文字が飛ぶ・回転・スライド）

### アニメーション・インタラクション
- **マウスインタラクション（カーソルに追従する独自カーソルデザイン）**
- ページトランジションが滑らかで映画的
- スクロール速度に応じてテキスト・画像がズレる（パーティクルスクロール）
- ホバー時の要素変形（画像のクロップ変化など）
- ローディングページあり（`#111111`背景に文字が現れる演出）

### レイアウト
- コンテンツを画面いっぱいに展開しながら、情報密度は低く保つ
- ナビゲーションはシンプル（ハンバーガーメニューまたはフローティング）
- コンタクトページも世界観を壊さないデザイン

### UXの特徴
- 「戦略的なブレインストーミングから形成されたワールドビュー」を体現
- ユーザーが次を見たくなるストーリー構造

## 3. MOMOTARO — https://momotaro.design/

**受賞歴:** Awwwards Honorable Mention / CSS Design Awards SOTD・WOTD・BEST PORTFOLIO SITE 2019 / FWA OF THE DAY  
**制作:** MOMOTARO（Yuto Takahashi）  
**タグ:** Clean, Graphic design, Minimal, Portfolio, Typography, Gallery, CSS, HTML5, jQuery

### ビジュアル・カラー
- **カラーパレット: `#2779a7`（ブルー）× `#FF9398`（ピンク）× `#ffffff`（白）**
- 淡い背景に鮮やかなアクセント2色という意外な組み合わせ
- 写真・作品画像を大きく扱うギャラリー型

### タイポグラフィ
- ミニマル・クリーン。余白を最大化したタイポグラフィ
- 日英テキストをシンプルに配置、主張しすぎない
- フォントはサンセリフ系、軽量なウェイトで統一

### アニメーション・インタラクション
- **マウスホバーで画像がビューポートに現れるギャラリーインタラクション**
- フィルター・エフェクト系のCSS表現
- スクロール連動の要素出現（シンプルなフェードイン）
- 過剰な演出はせず、インタラクションの「質」で勝負

### レイアウト
- 作品リストはテキストベースのナビゲーション形式
- 余白を大きく取ったグリッドレイアウト
- ポートフォリオとして「作品が主役」の構造が明確

### 特記事項
- シンプルに見えてインタラクションの精度が高い
- 「ミニマルだが退屈でない」バランスの手本

## 共通する設計思想

| 要素 | 共通パターン |
|------|-------------|
| カラー | ダーク or 強コントラスト。アクセントは1〜2色のみ |
| タイポグラフィ | 大きく・大胆に。日英混在。余白で重みを出す |
| アニメーション | スクロールトリガー + マウスインタラクション |
| ローディング | 独自ローディングで世界観を即座に伝える |
| ナビゲーション | ミニマル（ハンバーガー or フローティング） |
| レイアウト | フルスクリーン基本単位。グリッドより余白優先 |
| 技術スタック | GSAP / WebGL / Three.js / CSS Transitions |

## 現在のサイト（github/hub）への取り入れ方針

### 優先度高：すぐ取り入れられる要素
1. **カラーパレットの刷新** — 現状: GitHubデフォルトの白背景 → 変更: ダーク背景（`#0d0d0d`〜`#111`）+ アクセントカラー1色
2. **タイポグラフィの強化** — 欧文: Space Grotesk / DM Sans / Inter　日本語: Noto Sans JP / Zen Kaku Gothic New
3. **スクロールアニメーション** — GSAPのScrollTriggerでセクションの出現アニメーション / テキストの文字単位フェードイン（SplitType + GSAP）
4. **カスタムカーソル** — 小さなドット or リングがマウスに追従
5. **ナビゲーション刷新** — フローティング or ハンバーガーメニュー

### 優先度中：段階的に追加する要素
1. **ページトランジション** — ページ遷移時のオーバーレイアニメーション（Barba.js等）
2. **ホバーインタラクション** — 作品サムネイルのホバーエフェクト / ボタン・リンクのマグネティックエフェクト
3. **ローディング画面** — シンプルなカウンター or ロゴアニメーション

### 優先度低：将来的に検討
1. **WebGLエフェクト** — Three.js / GLSL によるパーティクルやディストーション（実装コストが高いため、他が安定してから）
